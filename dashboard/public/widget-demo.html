Product Update

Welcome to Acme Corp

This page simulates any customer website. The chat widget in the bottom-right corner is powered by SupportGenie. Click the purple bubble to start chatting!

How it works

The widget loads from a single script tag. It reads configuration (agent name, color, welcome message, API token) from data attributes and creates a full chat interface. Messages are sent to the SupportGenie Chat API.

Try it out

Click the chat bubble in the bottom-right corner. Type a question like "How do I reset my password?" to see it in action. If the Chat API (port 3001) is running, you'll get real AI-powered responses.
